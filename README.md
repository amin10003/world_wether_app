
# Weather Forecast App

Simple desktop weather application built using Python.

## Features

- Search weather by city
- Display temperature
- Save weather history
- View previous searches

## Built With

- Python
- CustomTkinter
- SQLite
- Requests API
- OOP

## Installation

Clone project

```
clone https://github.com/amin10003/world_wether_app.git

```

Create environment

```
python -m venv venv
```

Activate

Windows:

```
venv\Scripts\activate
```

Linux:

```
source venv/bin/activate
```

Install packages

```
pip install -r requirements.txt
```

Run

```
python main.py
```
